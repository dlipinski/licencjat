# licencjat

### Uruchomienie
Należy zainstalować:
1. NodeJS
2. npm
3. mongodb (i uruchomić : https://docs.mongodb.com/manual/tutorial/install-mongodb-on-ubuntu/) (inny system tez mozna sobie wybrac)

Potem:
```
git clone https://github.com/dlipinski/licencjat.git
cd licencjat
npm install
```

Teraz juź moźna uruchomić:
```
npm start
```
