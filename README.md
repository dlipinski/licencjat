# licencjat

### Uruchomienie
Należy zainstalować:
1. NodeJS
2. npm
3. mongodb

Potem:
```
git clone https://github.com/dlipinski/licencjat.git
cd licencjat
npm install
```

Teraz juź moźna uruchomić:
```
npm start
```
